## 📌 Features

* 🔍 Search and select any movie
* 🎯 Get top 10 similar movie recommendations
* 🎞️ Movie posters fetched using TMDB API
* ⭐ Ratings and overview for each movie
* ⚡ Fast performance with caching
* 🎨 Premium dark UI (OTT-style layout)

---

## 🧠 How It Works

This project uses a **content-based recommendation system**.

* Movies are represented using text features (tags)
* Similarity between movies is calculated using vectorization
* The system recommends movies based on similarity scores

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/yourusername/movie-recommender.git
cd movie-recommender
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Add your TMDB API key

Create a `.env` file or set environment variable:

```
TMDB_API_KEY=your_api_key_here
```

4. Run the app

```
streamlit run main.py
```
